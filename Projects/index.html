GVIDS：A Reliable Vehicle Intrusion Detection System Based on Generative Adversarial Network 2021/6 - 2022/3
· Designed an intelligent vehicular intrusion detection system based on GAN, which can not only detect all the current automobile injection attacks and Dos attacks, but also detect our newly proposed ×× attack.
· Responsible for the design and implementation of intrusion detection system, including scheme design, algorithm selection, network structure optimization, experimental evaluation, etc.

ClockIDS：A Real-time Vehicle Intrusion Detection System Based on Clock Skew 2021/2 - 2021/9
· Designed an intrusion detection system suitable for automobile CAN bus, and the detection rate was up to 98.39%; At the same time, it can also identify the device controlled by the attacker, with an average recognition rate of 96.41%.
· Responsible for system design and implementation, including scheme design, algorithm selection, experimental evaluation, etc.

VehicleEIDS: A Novel External Intrusion Detection System Based on Vehicle Voltage Signals 2020/9 - 2021/6 (Chinese Patent)
· Designed the vehicle-mounted external intrusion detection system, which is not limited by the means of packet injection by attackers, with an accuracy of up to 97%.
· Responsible for system implementation, including calculation and distribution selection and implementation, experimental evaluation, achievement protection, etc.
